## Подключение передатчика (Transmitter)

### Arduino Nano Pinout

| Компонент | Пин Arduino | Примечание |
|-----------|-------------|------------|
| **MPU-6050** | | |
| VCC | 3.3V | Важно: не 5V! |
| GND | GND | |
| SCL | A5 | I2C Clock |
| SDA | A4 | I2C Data |
| **NRF24L01+** | | |
| VCC | 3.3V | Важно: конденсатор 10µF рядом! |
| GND | GND | |
| CE | D9 | |
| CSN | D10 | |
| SCK | D13 | |
| MOSI | D11 | |
| MISO | D12 | |
| **Кнопка Home** | | |
| Пин 1 | D2 | |
| Пин 2 | GND | Внутренний pull-up |

### Важно

1. **NRF24L01+** требует хороший источник питания — добавь конденсатор 10µF между VCC и GND
2. **MPU-6050** только на 3.3V — на 5V сгорит
3. Для дальности используй NRF24L01+PA+LNA (с антенной)

### Расходы энергии

| Компонент | Ток |
|-----------|-----|
| MPU-6050 | ~3.8mA |
| NRF24L01+ | ~12mA (TX) / 13mA (RX) |
| Arduino Nano | ~15mA |
| **Итого** | ~40mA |

При питании от 18650 (2500mAh) — ~60 часов работы.
