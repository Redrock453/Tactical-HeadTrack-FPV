## Подключение оптоволоконного передатчика

### Вариант 1: Toslink модули (TOS173/TOR173)

**Бюджетный вариант ($2-5)**

```
Arduino Nano                TOTX173 (Передатчик)
-----------                -------------------
3.3V    ---+               VCC ---[220Ω]--- 5V
         |
         +--- MPU-6050 VCC         IN --- TX (D1)
GND     ---+--- MPU-6050 GND        GND --- GND
A5          --- MPU-6050 SCL
A4          --- MPU-6050 SDA

D2          --- Кнопка Home --- GND
```

### Вариант 2: Vishay VTLT1A00 (Прямой TTL)

**Более надежный ($5-10)**

```
Arduino Nano                VTLT1A00
-----------                ---------
5V      ------------------- VCC
GND     ------------------- GND
TX (D1) ---[220Ω]---------- ANODE
                           CATHODE --- GND
```

### Вариант 3: Broadcom HFBR-1521

**Профессиональный ($15-20)**

```
Arduino Nano                HFBR-1521
-----------                ----------
5V      ------------------- VCC
GND     ------------------- GND
TX (D1) ---[470Ω]---------- DATA IN
```

### POF Кабель

- **Тип:** Пластиковое оптоволокно 1mm (POF)
- **Длина:** 1-5м (зависит от применения)
- **Коннектор:** Toslink или custom разъемы

**Важно:**
- Минимальный радиус изгиба: 25mm
- Не перегибать кабель под 90°
- Защищать от механических повреждений

### Схема на Fritzing

```
                    +5V
                     |
                    [220Ω]
                     |
Arduino TX ---+------|-------> TOTX173 VCC
              |      |
              |     GND
              |
             GND

MPU-6050 подключен к I2C (A4/A5)
```

### Расходы энергии

| Компонент | Ток |
|-----------|-----|
| MPU-6050 | ~3.8mA |
| TOTX173 | ~15mA |
| Arduino Nano | ~15mA |
| **Итого** | ~34mA |

**Батарея 18650 (2500mAh):** ~73 часа работы
