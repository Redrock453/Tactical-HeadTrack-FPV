## Подключение оптоволоконного приемника

### Вариант 1: Toslink модули (TOS173/TOR173)

```
Arduino Nano                TORX173 (Приемник)
-----------                -------------------
5V      ------------------- VCC
GND     ------------------- GND
RX (D0) ---[0.1µF]---------- OUT

Серво Pan: D6 -> 5V BEC
Серво Tilt: D5 -> 5V BEC
```

### Вариант 2: Vishay VLR11

```
Arduino Nano                VLR11
-----------                ---------
5V      ------------------- VCC
GND     ------------------- GND
RX (D0) ------------------- OUT
           [10kΩ pullup]
             |
            5V
```

### Вариант 3: Broadcom HFBR-2521

```
Arduino Nano                HFBR-2521
-----------                ----------
5V      ------------------- VCC
GND     ------------------- GND
RX (D0) ------------------- DATA OUT
           [4.7kΩ pullup]
             |
            5V
```

### Подключение сервоприводов

```
                BEC 5V (2A+)
                 |
        +--------+--------+
        |                 |
     Серво Pan        Серво Tilt
        |                 |
      D6 (сигнал)      D5 (сигнал)
      GND              GND

Важно: Общий GND с Arduino!
```

### Полная схема

```
POF Кабель
    |
TORX173
    |----------- Arduino RX (D0)
    |           Arduino GND
    |
BEC 5V --------+----------- Серво Pan (сигнал D6)
               |           Серво Tilt (сигнал D5)
               |
            Arduino 5V (через VIN)
```

### Важно

1. **Уровень сигнала:** Оптоволоконные модули обычно работают с TTL 5V
2. **Фильтрация:** Конденсатор 0.1µF на входе RX для подавления шума
3. **Внешний BEC:** Сервы требуют отдельного источника питания
4. **Длина кабеля:** Toslink/POF до 10м надежно работает

### Отладка

- Светодиод на TORX173 должен моргать при приеме данных
- При обрыве кабеля — светодиод не горит
- Используйте Serial Monitor для проверки пакетов
