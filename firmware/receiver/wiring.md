## Подключение приемника (Receiver)

### Arduino Nano Pinout

| Компонент | Пин Arduino | Примечание |
|-----------|-------------|------------|
| **NRF24L01+** | | |
| VCC | 3.3V | Конденсатор 10µF рядом! |
| GND | GND | |
| CE | D9 | |
| CSN | D10 | |
| SCK | D13 | |
| MOSI | D11 | |
| MISO | D12 | |
| **Сервопривод Pan** | | |
| Сигнал (оранжевый/желтый) | D6 | |
| VCC (красный) | 5V | Отдельный источник! |
| GND (коричневый) | GND | |
| **Сервопривод Tilt** | | |
| Сигнал (оранжевый/желтый) | D5 | |
| VCC (красный) | 5V | Отдельный источник! |
| GND (коричневый) | GND | |

### Важно

1. **Сервоприводы** потребляют до 500mA каждый — не питай их от Arduino!
2. Используй внешний BEC (5V 2A+)
3. Общий GND для всех компонентов

### Питание дрона

```
LiPo Battery (3S-4S) -> BEC 5V -> Сервы
                           -> Arduino (через VIN или 5V)
                           -> NRF24L01+ (через 3.3V регулятор Arduino)
```
