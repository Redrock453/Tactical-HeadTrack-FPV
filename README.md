# Tactical-HeadTrack-FPV

> Отказоустойчивая система отслеживания головы для FPV с тактическим уклоном

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## О проекте

Система head tracking для управления камерой FPV дрона. Датчик на голове пилота → радиоканал → сервоприводы на камере.

**Тактические особенности:**
- Фильтрация Калмана/Маджвика для плавного движения
- Режим "Home" — мгновенный возврат в центр
- Опция оптоволоконной связи (защита от РЭБ)
- Интеграция с Android (Serial output)

## Железо

| Компонент | Модель | Примечание |
|-----------|--------|------------|
| Контроллер | Arduino Nano / Pro Mini | Компактный |
| Датчик | MPU-6050 | Гироскоп + акселерометр |
| Связь | NRF24L01+ / ESP32 | ESP-NOW для WiFi |
| Сервы | MG90S (2 шт) | Металлический редуктор |
| Кнопка | Тактовая | Режим Home |
