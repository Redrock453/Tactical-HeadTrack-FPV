# Оптоволоконная передача данных — Исследование

## Зачем оптоволокно?

| Преимущество | Описание |
|--------------|----------|
| **Защита от РЭБ** | Невосприимчивость к RF джаммингу |
| **EMI/RFI иммунитет** | Не боится помех от моторов/передатчиков |
| **Гальваническая развязка** | Нет общей земли — защита от скачков |
| **Безопасность** | Сигнал нельзя перехватить без физического доступа |

## Технологии

### 1. Toslink (S/PDIF) модули — Самый дешевый вариант

#### Компоненты

| Компонент | Описание | Цена |
|-----------|----------|------|
| **TOTX173** / **TOTX1771** | Передатчик 650nm, Toslink | $1-2 |
| **TORX173** / **TORX1771** | Приемник 650nm, Toslink | $1-2 |
| **POF кабель** | Пластиковое оптоволокно 1mm | $0.5/м |

#### Характеристики
- **Длина волна:** 650nm (красный)
- **Скорость:** До 12.5 Mbps (хватит для UART 115200)
- **Макс. длина:** 5-10м (POF)
- **Разъем:** Toslink (стандартный аудио)

#### Ограничения
- Нужна модификация схемы — Toslink модули designed для S/PDIF
- Формат данных: biphase mark encoding — нужен конвертер
- **Решение:** Использовать модули напрямую с TTL уровнями

#### Схема подключения

```
Arduino TX -> [Резистор 220Ω] -> TOTX VCC
TOTX GND -> GND

TORX VCC -> 5V
TORX GND -> GND
TORX OUT -> [Конденсатор 0.1µF] -> Arduino RX
```

### 2. Vishay VTLx / VLRx серия

| Модель | Тип | Скорость | Цена |
|--------|-----|----------|------|
| **VTLT1A00** | Tx (LED) | DC-10MBd | ~$3 |
| **VLR11** | Rx | DC-10MBd | ~$3 |
| **VTL1G** | Tx + наrada | DC-5MBd | ~$5 |

#### Преимущества
- Прямая TTL совместимость
- Более стабильная работа с UART
- Различные корпуса: through-hole, SMD

### 3. Broadcom HFBR-xxxx серия (Профессиональная)

| Модель | Тип | Скорость | Цена |
|--------|-----|----------|------|
| **HFBR-1521** | Tx 650nm | DC-5MBd | ~$10 |
| **HFBR-2521** | Rx 650nm | DC-5MBd | ~$10 |

#### Преимущества
- Индустриальное качество
- Широкий температурный диапазон
- Сертификации для авто/авиации

## Рекомендация для проекта

### Бюджетный вариант ($5-10)
```
Toslink TOTX173 + TORX173 + POF кабель
```

### Надежный вариант ($15-20)
```
Vishay VTLT1A00 + VLR11 + POF кабель
```

### Профессиональный вариант ($50+)
```
Broadcom HFBR-1521 + HFBR-2521 + качественный POF кабель
```

## POF против Glass Fiber

| Параметр | POF (Пластик) | Glass Fiber |
|----------|---------------|-------------|
| **Цена** | Очень дешевый | Дорогой |
| **Гибкость** | Отличная | Хрупкая |
| **Мин. радиус изгиба** | ~25mm | ~30mm |
| **Макс. длина** | 100м | Километры |
| **Температурный диапазон** | -40°C to +85°C | -40°C to +125°C |
| **Применение** | Авто, индустрия | Телеком |

**Вывод:** Для FPV (1-5м кабель) POF — идеальный выбор.

## Важные моменты

1. **Коннекторы** — Toslink разъемы можно купить готовые
2. **Обрезка кабеля** — Нужен special stripper и полировальный инструмент
   - Или покупать готовые кабели нужной длины
3. **Механическая защита** — Оптоволокно в гофротрубе для защиты от перегибов
