# Исправления безопасности v2.0

## 🔴 Критические исправления

### 1. Race condition в ISR
**Было:** `bool calibrateRequested = false;`
**Стало:** `volatile bool calibrateRequested = false;`

Добавлено `volatile` для правильной работы с прерываниями.

### 2. Валидация входных данных в receiver
**Было:** Прямая конвертация без проверки
```cpp
targetPan = (int)(rxData.yaw * 1.8f);
```

**Стало:** Функция `validateData()` с проверками
- Проверка чексуммы
- Проверка диапазона (yaw, pitch ≤ 100)
- Проверка флага home (0 или 1)

### 3. Добавлен XOR чексумма в NRF24 версию
Теперь оба варианта (radio и optical) используют проверку целостности данных.

## 🟡 Средние исправления

### 4. Watchdog Timer
Добавлен `wdt_enable(WDTO_2S)` во всех файлах.
Если код зависнет — автоматический перезапуск через 2 секунды.

### 5. Debounce для кнопки Home
Добавлен аппаратный фильтр с временным окном 50ms.

### 6. Magic Numbers заменены на константы
```cpp
const float ACCEL_SCALE = 16384.0f;  // ±2g range
const float GYRO_SCALE = 131.0f;    // ±250°/s range
const int SERVO_RANGE = 100;        // Диапазон передачи данных
```

### 7. Защита от переполнения буфера
Добавлена проверка `MAX_ERRORS` и очистка буфера при превышении.

## 🟢 Улучшения качества

### 8. Safe Servo Write
Функция `safeServoWrite()` гарантированно ограничивает значения.

### 9. Константы для валидации
```cpp
const uint8_t MAX_YAW_VALUE = 100;
const uint8_t MAX_PITCH_VALUE = 100;
```

### 10. Улучшенное чтение Serial в optical receiver
Проверка размера пакета перед обработкой.
